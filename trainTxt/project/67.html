Date: Tue, 26 Nov 1996 00:41:06 GMT Server: NCSA/1.5 Content-type: text/html Last-modified: Wed, 11 Sep 1996 17:34:00 GMT Content-length: 2638 IRIS Project IC-6: Intelligent Scheduling IRIS Project IC-6: Intelligent Scheduling at SFU The Intelligent Systems Lab (ISL) at Simon Fraser University is a participant in the IRIS project IC-6 on intelligent scheduling. The main research goal of the project is the synthesis of logic programming, constraint reasoning, belief revision, temporal reasoning and abductive reasoning into a viable intelligent scheduling framework. A more detailed description of the project is also available. There is also a brochure available in Adobe's PDF format. The participants in the project come from Canadian universities and companies. Some the ISL's project related documents are available.
