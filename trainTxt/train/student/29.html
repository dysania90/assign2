activities i am interested in making distributed computing accessible to common programmers i have coauthored the design and implemented a programming environment for workstation clusters it is unique in that it separates the programming model from the execution environment programs are written for a shared memory multiprocessor and applications execute on a network of dynamically changing workstations the load balancing fault masking and data coherence are provided transparently by the system the working prototype has demonstrated impressive performance results on a network of workstations with fluctuating workloads network traffic and failures for more information see the expanded version of my research activities or visit the official calypso home page thesis advisor
