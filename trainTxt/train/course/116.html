page advanced computer architecture description and goal computer science 580 and electrical and computer engineering 528 are currently combined to form an advanced computer architecture course that integrates a large laboratory exercise with a significant writing assignment the laboratory exercise requires small groups of students to understand or create several key tools used to study the performance of computer architecture and implementation techniques understand and modify the code generation portion of an ansi c compiler create an assembler capable of reading the code generated by the ansi c compiler and generating an executable file develop code optimization tools create a loader capable of loading the executable image into a simulated memory image construct an instruction level cpu simulator capable of executing the loaded image use pipelining super pipelining superscalar or vliw implementation techniques to obtain good overall performance in terms of cycles per instruction cpi the writing assignment is a single final report that will be carefully written and edited to describe the products constructed during the laboratory exercise in addition this document will describe the performance of the final system and point out all important assumptions and simplifications the goal of this course is to give each student a realistic design experience where each team is responsible for the decisions they make and the quality of their work in addition the format of the class is very similar to the research process of choosing a topic doing the research or work
