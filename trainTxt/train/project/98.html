department courant nyu persistent linda a software system for robust distributedparallel computing want to write to us objectives the objectives of the plinda project are twofold fault tolerant parallel computing provide language constructs that facilitate the design of high performance faulttolerant parallel programs harness the aggregate power of idle cycles on networks of workstations in a manner that is faulttolerant and does not disturb the owners of the workstations at all our system models nonidleness eg user returns to workstation as failure the work being done at the idle workstation is then moved applications the major class of plinda applications are longrunning coarse grain parallel applications example applications nile project data mining eg biological pattern
